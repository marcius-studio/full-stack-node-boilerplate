# Vue, Koa, MongoDB full-stack boilerplate

Minimal build of front-end and back-end for quick development start.

## Features

**Client-side**

* [x] Vuejs 2.x
* [x] @vue/cli-service (Webpack 4, SASS/SCSS)

**Server-side**

* [x] Koa
* [x] Koa router
* [x] MongoDB as Mongoose
* [x] Nodemon

## Commands

Build folder `dist` for both sides.

```node
// client\package.json
$ npm run serve
$ npm run build

// server\package.json
$ npm run start
$ npm run build
```
